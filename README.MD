# Мир кораблей на macOS

Данная сборка позволяет играть в Мир кораблей без виртуальной Windows на macOS в 1 клик с процессором Apple silicon.


Видео с геймплеем:

#### [YouTube](https://youtu.be/Kku1NjYM3EY)

#### [Rutube](https://rutube.ru/video/0ca47163b7ee6301ed33ffc45ca96d5b/)

## Установка
**только для Apple silicon**

Если у вас уже установлена предыдущая сборка, то новая перекинет корабли из старой и перекачивать не придётся

Скачайте и следуйте инструкциям установщика
